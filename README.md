# ふらっとん
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
Flappy Birdがモチーフ。こうかとんが土管に当たらないようにジャンプさせるゲーム

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス+描画
* 土管に関するクラス+描画
* 背景描画
### 担当追加機能
* 点数追加: 土管を通過した際に1ポイント獲得
* 一定点数で無敵(+表示変更): 50ポイントとか取れたら無敵
* コイン機能: コイン集め　浮いてる
* 良いアイテム機能: 移動速度が低下
* 悪いアイテム機能: 移動速度が上昇
